\# Face Recognition Attendance System



A Python-based Face Recognition Attendance System that uses OpenCV and the `face\_recognition` library to detect faces in real-time using a webcam and automatically mark attendance in a CSV file.



\---



\## 🚀 Features



\- Real-time face detection using webcam

\- Face recognition using pre-stored images

\- Automatic attendance marking

\- Stores attendance with timestamp in CSV file

\- Simple and beginner-friendly implementation



\---



\## 🛠️ Tech Stack



\- Python

\- OpenCV

\- face\_recognition

\- NumPy

\- CSV module
